# 🎨 Word Art Image Fill

A powerful web application for creating stunning text art with image fills. Upload multiple images, customize fonts, and export your creations as PNG files.

## ✨ Features

- 📝 **Custom Text**: Write any word up to 30 characters
- 🎨 **8 Font Styles**: Choose from popular fonts
- 📏 **Dynamic Sizing**: Scale text from 50-300px
- 🖼️ **Multiple Image Support**: Upload and switch between multiple images
- 🎯 **Image Navigation**: Easily navigate through your uploaded images
- 🔄 **Precise Positioning**: Fine-tune image placement with X and Y offset controls
- 💾 **Export as PNG**: Save your artwork in high quality
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices
- ⚡ **Real-time Preview**: See changes instantly as you adjust settings

## 🚀 Usage

1. **Enter Your Word**: Type any word up to 30 characters
2. **Choose Font Style**: Select from 8 different font options
3. **Adjust Font Size**: Use the slider to scale your text (50-300px)
4. **Select Fill Mode**: Choose between per-word or per-letter fill
5. **Upload Images**: Select multiple images from your device
6. **Navigate Images**: Use Previous/Next buttons to switch between images
7. **Position Image**: Adjust X and Y offset sliders to place the image perfectly
8. **Save**: Click "Save as PNG" to download your creation

## 🛠️ How to Deploy on GitHub Pages

### Option 1: Using GitHub Desktop
1. Create a new repository called `word-art-image-fill`
2. Clone it to your computer
3. Copy all the files (index.html, styles.css, script.js, README.md) to the cloned folder
4. Commit and push the changes
5. Go to repository Settings → Pages
6. Select "Deploy from a branch" and choose `main` branch
7. Your app will be live at `https://yourusername.github.io/word-art-image-fill`
